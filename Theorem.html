Properties of conjugate

Theorem : If z1, z2 are complex numbers, then

(i) \( \bar{ ( {\bar{z}} ) } \) = z

(ii) z + \(\bar{z} \) = 2Re(z)

(iii) z - \(\bar{z} \) = 2iIm(z)

(v) z + \(\bar{z} \) = 0 ⇒ z is purely imaginary

(vi) z\(\bar{z} \) = Re(z)2 + Im(z)2
