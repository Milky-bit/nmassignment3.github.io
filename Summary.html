Summary
  • If a,b are real numbers, then a number z = a + ib$ is called a complex number, real number a is known as the real part of zand b is known as its imaginary part. We write a = Re(z), b = Im(z) .

    A complex number z is purely real iff Im(z) = 0 and z is purely iff Re(z) = 0.

  • The conjugate of a complex number z = a + ib is denoted by \( \bar{z} \) and is equal to a - ib. For any three complex numbers z, z1, z2, we have

    (i) \( \bar{ ( {\bar{z}} ) } \) = z

    (ii) z + \(\bar{z} \) = 2Re(z)

    (iii) z - \(\bar{z} \) = 2iIm(z)

    (iv) z = \(\bar{z} \) ⇔ z is purely real

    (v) z + \(\bar{z} \) = 0 ⇒ z is purely imaginary

    (vi) z\(\bar{z} \) = Re(z)2 + Im(z)2

  • The modulus of a complex number z= a + ib is denoted by |z| and is defined as

    |z| = √(a2 + b2) =√(Re(z)2 + Im(z)2)

    If z, z1, z2 are three complex numbers, then

    • (i) |z| = 0 ⇔ z = 0 i.e. Re(z) = Im(z) = 0

    (ii) |z| = |\(\bar{z} \)| = |-z|

    (iii) -|z|≤ Re(z)≤ |z| ; -|z| ≤ Im(z) ≤ |z|

    (iv) z\(\bar{z} \) = |z| 2

    (v) |z1 z2| = |z1| |z2|

    (vi) | z1 ⁄ z2| = | z1| ⁄ |z2|

    (vii) |z1 + z2|2 + |z1 - z2|2 = 2(|z1|2 + |z2|2)

    (viii) |az1 - bz2|2 + |bz1 + az2|2 = (a2 + b2) (|z1|2 + |z2|2), where a,b ∈ R.
